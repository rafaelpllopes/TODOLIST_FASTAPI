# To Do List
Será criado uma aplicação simples de gerenciamento de tarefas (To-Do List). A aplicação deve permitir que os usuários criem, leiam, atualizem e excluam tarefas.

## Ambiente

- Ubuntu
- Python 3.12
- Docker
- MariaDB
- Git
